Thank You For Taking a Look At My School Work

The Following are some of the interesting assignments we worked on in professor William Kerney's Discrete Mathematics and Data Structures class.

Binary Search Tree

Our assignment was to create a Binary Search Tree and implement the following functions: "Insert, Search, Delete and Print".

Inventory

Our assignment was to implement different aspects of the Set Theory as a way of understanding how to work with data in a Database. Specifically we had to implement a way to read SQL like commands such as "Insert, Join, Inner Join and Select From" and then carry out the commands.

Linked List

Our assignment was to create a linked list and implement the following functions: "Pop, push, search, and Remove".

Merge Sort Golf

This was a submission for a class wide competition to implement a merge sort while using the least possible numbers of semicolons. I was able to do it with just "three"!

Recursion Assignment

As a way of mastering Recursion this assignment required us to create a program that takes in a number and creates a diamond of that size.

Queues And Stacks

We had to implement a queue using an array to gain an understanding of how they work.
